# ESP32-MacroKeyboard

Умная макроклавиатура на ESP32-S3 с Wi-Fi. Эмулирует клавиатуру и мышь через USB HID. 10-13 кнопок выполняют макросы, управление через веб-интерфейс.

## 📋 Возможности

- 🖱️ Эмуляция клавиатуры и мыши (USB HID)
- 📡 Wi-Fi точка доступа для управления
- 🌐 Веб-интерфейс (http://10.0.0.1)
- 🔘 10-13 программируемых кнопок
- ⌨️ Макросы: клавиши, комбинации, задержки
- 🖱️ Управление мышью: движение, клики
- 📝 Текстовые вставки
- ⚡ Автоматизация рабочих процессов

## 🔧 Оборудование

- **Плата:** YD-ESP32-23 (ESP32-S3 N16 R8)
- **Кнопки:** пины 4, 5, 6, 7, 8, 9, 15, 16, 17, 18, 3, 10, 46
- **USB:** HID клавиатура + мышь
- **Wi-Fi:** точка доступа 10.0.0.1
